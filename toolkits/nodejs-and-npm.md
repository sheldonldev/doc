# NodeJS and npm


## Installation

```bash
brew install node
npm install -g yarn
npm install -g browser-sync
```


## Update

```bash
# with homebrew
brew upgrade node
npm install -g npm

# with npm
sudo npm cache clean -f
sudo npm install -g n
sudo n stable

node -v
npm -v
```
