# Basic Commands

## Start

```bash
vim -o file1 file2 file3
# -o is horizon
# -O is vertical
```

## Windows

```bash
:new
:vne[w]
:vs[plit]

:h opening-window
```

```bash
:h window-moving
```

```bash
:h window-resize
```

## Search Files

```bash
:Rg           # regex
:find _file_
:b _file_
:ls           # buffer list
```

## netrw

- It's a built in plugin now

```bash
:h netrw
```
